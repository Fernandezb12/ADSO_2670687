Concepto de GIT

Git es un sistema de control de versiones increible que podemos utilizar para gestionar y controlar cambios en proyectos para nosotros los desarrolladores de software. Ademas Git nos permite realizar un seguimiento de los cambios en los archivos de un proyecto a lo largo del tiempo. Con Git, se pueden crear diferentes versiones de un proyecto, realizar cambios experimentales en ramas separadas, entre otras maravillosas herramientas

PRINCIPALES COMANDOS

  • GIT INIT: Este comando funciona para iniciar git

  • GIT ADD: Este comando funciona para agregar todos los documentos al respositorio

  • GIT PUSH: Este comando funciona para enviar los cambios locales de un repositorio Git al repositorio remoto

  • GIT STATUS: Este comando funciona para mostrar el estado actual de los archivos en el respositorio

  • GIT COMMIT: Este comando funciona para comentar en el respositorio

  • GIT BRANCH: Este comando funciona para mostrar una lista de todas las ramas en el repositorio

  • GIT CONFIG --GLOBAL USER.NAME: Establece el nombre de usuario global que se utilizará en todos los repositorios Git en el sistema.

  • GIT CONFIG --GLOBAL USER.EMAIL: Establece la dirección de correo electrónico global que se asociará con los commits en todos los repositorios Git en el sistema.

  • GIT CHECKOUT:Este comando se utiliza para cambiar a una rama específica en el repositorio.

  • GIT LOG: El comando muestra el historial de commits en un repositorio Git.
